# Verda-challenge [![Build Status](https://travis-ci.org/nik-m2/Verda-challenge.svg?branch=master)](https://travis-ci.org/nik-m2/Verda-challenge)

## Project setup
```
npm install
```

## Usage

#### Development
```
npm run serve
```

Then go to localhost:8080

#### Production
```
npm run build
```
Output can be found in `build` folder

## Testing

#### Unit tests
```
npm test
```

#### Integration tests
```
npm run test:e2e
```
